## 1. 三大财务报表概述

| 报表 | 反映内容 | 核心指标 |
|------|----------|----------|
| 利润表 | 盈利情况 | 营收、净利润、毛利率、净利率 |
| 资产负债表 | 财务状况 | 总资产、总负债、净资产 |
| 现金流量表 | 现金流动 | 经营现金流、投资现金流、筹资现金流 |

### 财务报表之间的关系

```
净利润 = 营收 - 成本 - 费用 - 税
净资产 = 总资产 - 总负债（资产负债表）
经营现金流 ≠ 净利润（权责发生制 vs 收付实现制）
```

## 2. 关键财务指标

### 盈利能力指标

| 指标 | 公式 | 优质标准 |
|------|------|----------|
| ROE（净资产收益率） | 净利润/净资产 | > 10%，越高越好 |
| 毛利率 | (营收-成本)/营收 | > 20% |
| 净利率 | 净利润/营收 | > 10% |

### 财务风险指标

| 指标 | 公式 | 优质标准 |
|------|------|----------|
| 资产负债率 | 总负债/总资产 | < 70% |
| 流动比率 | 流动资产/流动负债 | > 1.5 |

### 运营效率指标

| 指标 | 公式 | 说明 |
|------|------|------|
| 存货周转率 | 销货成本/平均存货 | 越高变现越快 |
| 应收账款周转率 | 营收/平均应收账款 | 越高回款越快 |

## 4. 多股票财务对比

```python

stock_list = [
    ("000001", "平安银行"),
    ("600000", "浦发银行"),
    ("600036", "招商银行"),
    ("601166", "兴业银行"),
]

results = []
for code, name in stock_list:
    df = ak.stock_financial_analysis_indicator(symbol=code)
    df = df.sort_values('日期', ascending=False)
    latest = df.iloc[0]
    results.append({
        '股票代码': code,
        '股票名称': name,
        'ROE(%)': latest.get('净资产收益率(%)', None),
        '毛利率(%)': latest.get('销售毛利率(%)', None),
        '净利率(%)': latest.get('销售净利率(%)', None),
        '资产负债率(%)': latest.get('资产负债率(%)', None),
```
